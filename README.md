# 🦅 Club Atlético Deportivo Acarigua - Sistema de Gestión Deportiva

Este proyecto es una aplicación web integral diseñada para la recopilación, monitoreo y análisis antropométrico del rendimiento deportivo de los atletas del **Club Atlético Deportivo Acarigua**.

## 📖 Descripción del Proyecto

El sistema permite centralizar toda la información técnica y médica del club, facilitando el seguimiento del progreso físico de los jugadores a través de mediciones periódicas, control de asistencias y gestión de planes de entrenamiento.

### 🌟 Características Principales

- **Gestión de Atletas:** Registro detallado de deportistas con información personal, técnica y de contacto.
- **Monitoreo Antropométrico:** Seguimiento de peso, altura, porcentaje de grasa y otros índices de rendimiento.
- **Control de Asistencias:** Registro diario de presencia en entrenamientos y partidos.
- **Ficha Médica:** Historial de salud, alergias y condiciones físicas especiales.
- **Gestión de Personal:** Administración de entrenadores, médicos y personal directivo con roles específicos.
- **Módulo de Actividades:** Planificación de microciclos, mesociclos y actividades diarias.
- **Seguridad:** Sistema de permisos basado en roles (RBAC) y preguntas de seguridad para recuperación de cuenta.

## 🛠️ Tecnologías Utilizadas

- **Frontend:** [Vue.js 2](https://vuejs.org/) con [Element UI](https://element.eleme.io/)
- **Backend:** [Node.js](https://nodejs.org/) con [Express](https://expressjs.com/)
- **Base de Datos:** [MySQL](https://www.mysql.com/)
- **Autenticación:** [JSON Web Tokens (JWT)](https://jwt.io/)
